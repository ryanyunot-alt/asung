☁️ 클라우드 동기화 (Google Drive)

위쪽 "Google Drive 자동 동기화" 섹션에서 설정하세요. 한 번 연결하면 PC와 폰에서 자동으로 동일한 데이터를 사용할 수 있습니다.
설정 방법은 별도 가이드(Google_Drive_설정가이드.html)를 참고하세요.

📌 폰에서 앱처럼 쓰는 법

아이폰: Safari로 이 파일 열기 → 공유 버튼 → "홈 화면에 추가"
안드로이드: Chrome으로 열기 → 메뉴(⋮) → "홈 화면에 추가"

💾 데이터 저장 방식

모든 정보는 브라우저(이 기기)에 자동 저장됩니다. Google Drive 동기화를 켜면 클라우드에도 자동 저장됩니다.

📨 알림 발송 방식

미납자 알림 버튼을 누르면 휴대폰 기본 문자/메일 앱이 자동으로 열리면서 메시지가 미리 작성되어 있습니다. 확인 후 "보내기"만 누르시면 됩니다.
